Mengenal Saya Lebih Dekat

Saya, Andhika Rafi, adalah seorang pelajar yang bersemangat di bidang teknologi informasi, khususnya dalam jaringan komputer dan keamanan siber. Sebagai siswa kelas 11 dengan jurusan TJKT di SMK Wikrama Bogor, saya aktif mengembangkan keterampilan teknis saya melalui berbagai proyek nyata.

Minat saya mencakup Linux Server untuk pengelolaan infrastruktur server yang andal, Cloud Computing menggunakan platform AWS, serta Web Development untuk membangun aplikasi web yang fungsional dan modern.
