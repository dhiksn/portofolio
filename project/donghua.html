Donghua Streaming

Platform streaming donghua (anime China) berbasis web yang mengambil konten secara real-time dari anichin.run menggunakan web scraper dengan Cheerio. Backend dibangun dengan Node.js dan Express yang menyediakan REST API untuk frontend. Fitur meliputi halaman home, ongoing, completed, pencarian, filter genre, jadwal tayang, dan video player terintegrasi.

Fitur Utama

🕷️
Web Scraper Otomatis
Mengambil data anime secara real-time dari anichin.run menggunakan Cheerio tanpa perlu database.

Video Player Terintegrasi
Player video langsung di dalam halaman tanpa redirect ke sumber eksternal.
🔍
Pencarian & Filter Genre
Cari judul anime dan filter berdasarkan genre untuk menemukan konten yang diinginkan.
📅
Jadwal Tayang
Menampilkan jadwal episode terbaru dari anime yang sedang ongoing setiap minggunya.
🌐
REST API Backend
Backend Express.js yang menyajikan data melalui REST API untuk dikonsumsi frontend.
📂
Halaman Ongoing & Completed
Daftar anime yang masih tayang dan yang sudah selesai, diperbarui secara otomatis.
